# Get-a-Job

A career management platform built on React + Supabase. Helps job seekers track applications, analyse role fit, identify skill gaps, and prepare for interviews through AI-powered tools and a structured workflow.

---

## Features

- **AI Career Coach** — Chat with specialised AI agents (career coach, CV helper, interview prep, networking advisor)
- **Job Match Analysis** — Paste a job description and get a fit score with matched/missing requirements
- **Career Roadmap** — AI-generated tiered role recommendations based on your skills and goals
- **Tailored CV Generation** — AI-powered CV tailoring for specific job descriptions
- **Smart Task Management** — Personalised weekly action plans generated by AI
- **Application Tracker** — Track applications through the full pipeline (interested → applied → interview → offer)
- **Learning Paths** — AI-recommended courses and certifications to close skill gaps
- **Resume Upload** — Upload your CV for AI-powered data extraction during onboarding

---

## Getting Started

### Prerequisites
- Node.js 18+
- A Supabase project

### Environment Variables

Create a `.env.local` file in the project root:

```
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Additionally, set the `OPENAI_API_KEY` as a **Supabase project secret** (Dashboard → Edge Functions → Manage Secrets) for AI features.

> Never commit `.env.local`. It is listed in `.gitignore`.

### Install & Run

```bash
npm install
npm run dev        # development server at localhost:5173
npm run build      # production build
npm run preview    # preview production build locally
npm run lint       # check for lint errors
npm run lint:fix   # auto-fix lint errors
```
